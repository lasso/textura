    getSessionScope() : string

    The session scope can be one of three predefined values: 'global' When using this value, the session cookie associated with the request will have the path '/'. 'application' When using this value, the session cookie associated with the request will have the path returned by PathBuilder::getTexturaBaseURL(). 'controller' When using this value, the session cookie associated with the request will have the path returned by PathBuilder::buildRoute($current_controller_class).

    Exceptions

    \LogicException If sessions are not used for the current controller
