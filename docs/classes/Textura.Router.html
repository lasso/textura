    route(\Textura\Request $request, \Textura\Response $response) : null
    Static

    Rules for routing:

    1) If an exact match is found, the specified action is called directly 2) If no exact match is found, the "default" action for the current controller is called 3) If no default action is found and the path differs from /, the "parent" path is called 4) Repeat
