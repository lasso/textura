    Class that represents a user session

    Magic getter.

    __get(string $key) : mixed

    If key is 'session_id' the session id is returned. If key is 'session_name' the session name is returned. For any other keys, checks the currect session data and returns the current value. If key cannot be found in current session data, returns null.
