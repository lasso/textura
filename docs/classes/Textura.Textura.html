    registerPlugin(string $name, string $plugin_dir, string $plugin_file, string $plugin_class, string $plugin_namespace) 

    Parameters

    $name

    string

    Name of plugin to load.

    $plugin_dir

    string

    custom directory where plugin can be found. If null, uses TEXTURA_PLUGIN_DIR/$name.

    $plugin_file

    string

    custom file name where plugin can be found. If null, uses plugin_dir/$name.php.

    $plugin_class

    string

    custom class name that should be loaded. If null, uses "{$name}Plugin".

    $plugin_namespace

    string

    custom namespace that should be used to initialize the plugin class. If null, the "Textura" namespace is assumed.

    Exceptions

    \LogicException if the plugin cannot be loaded for some reason.
