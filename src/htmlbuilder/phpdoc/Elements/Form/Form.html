Form

Description
Description | Vars (details) | Methods (details) Constants (details)

The <form> tag is used to create an HTML form for user input.

A form can contain input elements like text fields, checkboxes, radio-buttons, submit buttons and more. A form can also contain select menus, textarea, fieldset, legend, and label elements. Forms are used to pass data to a server. http://www.w3schools.com/tags/tag_form.asp
