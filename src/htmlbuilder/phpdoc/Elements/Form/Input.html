Input

Description
Description | Vars (details) | Methods (details) Constants (details)

An input field can vary in many ways, depending on the type attribute.

An input field can be a text field, a checkbox, a password field, a radio button, a button, and more. http://www.w3schools.com/tags/tag_input.asp
