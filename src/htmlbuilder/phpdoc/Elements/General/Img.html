Img

Description
Description | Vars (details) | Methods (details) Constants

The <img> tag embeds an image in an HTML page.

Notice that images are not technically inserted into an HTML page, images are linked to HTML pages. The <img> tag creates a holding space for the referenced image. The <img> tag has two required attributes: src and alt. http://www.w3schools.com/tags/tag_img.asp
