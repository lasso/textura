Table

Description
Description | Vars (details) | Methods (details) Constants (details)

The <table> tag defines an HTML table.

A simple HTML table consists of the table element and one or more tr, th, and td elements. The tr element defines a table row, the th element defines a table header, and the td element defines a table cell. A more complex HTML table may also include caption, col, colgroup, thead, tfoot, and tbody elements. http://www.w3schools.com/tags/tag_table.asp
