  3.  * The <object> tag is used to include objects such as images, audio, videos, Java applets, ActiveX, PDF, and Flash.
  4.  * 
  5.  * The object element was intended to replace the img and applet elements.
  6.  * However, because of bugs and a lack of browser support this has not happened.
  7.  * The object support in browsers depend on the object type. Unfortunately, the
  8.  * major browsers use different codes to load the same object type. Luckily, the
  9.  * object element provides a solution. If the object element is not displayed,
  10.  * the code between the <object> and </object> tags will be executed.
  11.  * This way we can have several nested object elements (one for each browser).
